# 🔐 React Native | Autenticação com Supabase

Projeto desenvolvido utilizando **React Native** e **Supabase**, demonstrando como implementar um sistema completo de autenticação para aplicações mobile.

A integração com o Supabase permite criar um backend moderno com autenticação segura, gerenciamento de usuários e persistência de sessões, reduzindo significativamente a complexidade da infraestrutura.

---

## ⚙️ Configuração

Crie um projeto no **Supabase** e configure as variáveis de ambiente da aplicação.

```env
SUPABASE_URL=your_project_url

SUPABASE_ANON_KEY=your_anon_key
```
